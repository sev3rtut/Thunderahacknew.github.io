# ⚡ Thunderhacknew - Game Client Website

Стильный одностраничный сайт для игрового клиента с темным дизайном в фиолетовых тонах, вдохновленный VenusWare.

## 🚀 Особенности
* **Современный UI:** Адаптивный дизайн с неоновыми акцентами.
* **Admin Panel:** Встроенная возможность загрузки файлов (через PHP).
* **Ready to use:** Настроенная структура для быстрой загрузки на хостинг.
* **Icons:** Используется FontAwesome для визуализации функций.

## 📂 Структура проекта
* `index.html` — Главная страница сайта.
* `style.css` — Стили, градиенты и анимации.
* `admin.php` — Панель администратора для обновления файла клиента.
* `uploads/` — Папка, куда будут сохраняться твои файлы.
